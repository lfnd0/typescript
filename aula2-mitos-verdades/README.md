- Vou ter que aprender tudo de novo.
  - *Mito!*

  - *Mito!*
- Já escrevo testes, não preciso disso.
  - *Mito!*

  - *Mito!*
- O TypeScript é verboso demais, precisa tipar tudo.
  - *Mito!*
- Só serve para "projeto grande".
  - *Mito!*
- É só usar `PropTypes` que dá na mesma.
