## Comandos importantes

### Instalação das dependências:
> yarn

### Compilação do código:
> yarn tsc nome-do-aquivo.ts | --watch

Ou, com o `tsconfig.json` configurado:

> yarn tsc | --watch

### Execução do código:
> yarn dev
